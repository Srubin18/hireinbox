# HIREINBOX - Claude Code Context

**Mission:** AI-powered CV screening for South African SMEs
**Stack:** Next.js 16 + TypeScript + Supabase + OpenAI

## Quick Rules
- Use inline CSS (no Tailwind)
- SA-specific: CA(SA), BCom, local companies
- POPIA compliant - evidence-based AI
- Use `/src/lib/sa-context.ts` for AI features

## Current Status
- Live: Fine-tuned CV screening + Video analysis
- Pilot: `/pilot` pages for recruiters

## Key Locations
- `/src/app/pilot/` - Pilot pages
- `/src/app/api/` - API routes
- `/src/lib/` - Utilities

## Commands
- `npm run dev` - Dev server (port 3002)
- `/clear` - Clear conversation to save tokens
